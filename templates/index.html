    О проекте

    Этот проект использует технологию CLIP для семантического поиска изображений. Вы можете искать фотографии, описывая их содержимое на естественном языке.

    • Поддержка HEIC формата
    • Синхронизация с iCloud
    • Семантический поиск
    • Автоматическая индексация

    Как использовать

    Для поиска изображений просто введите описание того, что хотите найти. Например:

    • "закат на пляже"
    • "человек в красной куртке"
    • "горный пейзаж"

    Чем точнее описание, тем лучше будут результаты поиска.
