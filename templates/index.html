О проекте

Этот проект использует технологию CLIP для семантического поиска изображений. Вы можете искать фотографии, описывая их содержимое на естественном языке.

  • Поддержка HEIC формата
  • Синхронизация с iCloud
  • Семантический поиск
  • Автоматическая индексация

Как использовать

Для поиска изображений просто введите описание того, что хотите найти. Важно: поиск работает только на английском языке! Например:

  • "sunset on the beach"
  • "person in red jacket"
  • "mountain landscape"

Чем точнее описание на английском языке, тем лучше будут результаты поиска.
